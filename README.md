# Пример реализации стриминга на сайт
Как это будет выглядеть: https://cam1.yuccastream.com/hls.html

## Зависимости
 - docker 
 - docker-compose

## Пример использования

По умолчанию настроенно на парковку вебки ноутбука

запуск
```bash
docker-compose up -d
```

остановка
```bash
docker-compose down -v
```
